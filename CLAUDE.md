
# Purpose

- For teaching and learning basics of local development using MLX.

# Style

- Centralize variables like `model_name` at the top of files.
- Single files, minimal imports, extremely clear and concise. 
- Do not modify any files in the `examples` directory. They are learning examples for building up intuition.

# Usage

- Launch and run in a terminal on an Apple MacOS laptop.
- Prefer uv to run python.
